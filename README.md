  ## Description
  This is a CMS (contant management system) to allow non-developers to easily view and interact with information stored in databases.

  ## Installation
```
  npm install
 ```
 
 ## Usage

  run mySQL first by using the code below:
  ```
  mysql -u root -p
  #password#
  source schema.sql
  source seeds.sql
  ```
  then run
  ```
  node index
  ```
